**Modulplaner**
Das ist das Abschlussprojekt zum Modul Webtechnologien im Studiengang Angewandte Informatik.

Die Aufgabe war es auf Basis einer API-Request die sämtliche Daten von Modulen beinhaltet
übersichtlich darzustellen und einen Stundenplan daraus erstellen zu können.

Dafür durfte kein Framework benutzt werden, ausschließlich JavaScript (Node.js), HTML, CSS durften verwendet werden.
